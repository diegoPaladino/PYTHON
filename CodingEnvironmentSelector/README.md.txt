# CodingEnvironmentSelector

## Introdução
Este programa Python oferece uma interface simples para escolher entre ambientes de desenvolvimento para Arduino e Python. Ele abre o IDE do Arduino ou o Visual Studio Code, junto com arquivos de texto específicos, dependendo da escolha do usuário.

## Materiais Necessários
- Python 3 instalado.
- Arduino IDE e Visual Studio Code instalados nos caminhos especificados no script.

## Propósito do Projeto
Facilitar a transição entre diferentes ambientes de codificação, aumentando a eficiência e organização do desenvolvedor.

## Prós e Contras
Prós:
- Automatiza a abertura de ambientes de desenvolvimento e arquivos relacionados.
- Interface simples e fácil de usar.

Contras:
- Especificado para caminhos e arquivos pré-definidos, necessitando de ajustes para diferentes usuários.

## Diretrizes Gerais
Execute o script em um ambiente Python. Uma janela de diálogo aparecerá pedindo para escolher entre Arduino e Python. Após a escolha, o respectivo IDE e arquivo de texto serão abertos automaticamente.
