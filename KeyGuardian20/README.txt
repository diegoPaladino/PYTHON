# KeyGuardian20

## Introdução
O KeyGuardian20 é um sistema eletrônico de gerenciamento de chaves utilizando um Raspberry Pi. Ele suporta até 20 chaves e inclui um display LCD 24X2 e um teclado numérico para interação do usuário.

## Materiais Necessários
- Raspberry Pi (Modelo 3B+ ou superior)
- Display LCD 24x2 com módulo I2C
- Teclado numérico
- Fonte de alimentação para Raspberry Pi (5V, 2.5A)
- Gabinete para claviculário com 20 ganchos
- Fios de conexão
- Cartão SD com Raspbian instalado
- Fechadura eletrônica (opcional)

## Propósito
Facilitar a gestão de chaves em ambientes que necessitam de um controle de acesso seguro e eficiente, como escritórios, escolas e condomínios.

## Prós
- Acesso controlado por usuário e senha.
- Registro de atividades para monitoramento de acessos.
- Fácil de expandir e customizar.

## Contras
- Requer configuração inicial detalhada.
- Dependente de energia elétrica.

## Diretrizes Gerais
1. Monte o hardware conectando o display LCD e o teclado numérico ao Raspberry Pi.
2. Instale o sistema operacional Raspbian e as bibliotecas necessárias.
3. Implemente o sistema de gestão de usuários em Python.
4. Teste todas as funcionalidades.
