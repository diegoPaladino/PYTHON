# PriceTrackerPythonista

## Introduction
PriceTrackerPythonista is a Python-based web scraping tool designed to monitor and track online product prices. It allows users to keep an eye on price changes for specific products, aiding in smart shopping decisions.

## List of Necessary Materials
- Python 3.x
- Libraries: requests, bs4 (BeautifulSoup)
- Internet access

## Project Purpose
The primary purpose of this project is to provide a simple yet effective way to track the prices of products online. It's intended for shoppers who want to get the best deals by monitoring price fluctuations.

## Pros and Cons
### Pros:
- Automated price tracking
- Easy to use and set up
- Customizable for different products

### Cons:
- Specific to Amazon product layout
- Requires manual URL input
- Dependent on website structure

## General Guidelines
1. Install Python 3.x and necessary libraries (`requests`, `bs4`).
2. Run the script with a valid product URL from Amazon.
3. The script will output the current price and product title.
